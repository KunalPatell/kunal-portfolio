# Static assets

Drop these files here (they're referenced from `src/lib/data.ts`):

- **`resume.pdf`** - your resume. Until this file exists, the resume buttons request the latest copy by email.
- **`projects/`** - project cover assets used by the featured project cards.
- **`favicon.ico`**, social/OG image — optional polish.

Anything in this folder is served from the site root (`/resume.pdf`, etc.).
